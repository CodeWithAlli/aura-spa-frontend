# Aura Spa 🌿

Plataforma web moderna para spa de belleza. React + Vite + TailwindCSS.

## Requisitos

- Node.js v18+
- npm

## Instalación

```bash
npm install
```

## Desarrollo local

```bash
npm run dev
```

Abre http://localhost:5173 en tu navegador.

## Build para producción

```bash
npm run build
npm run preview
```

## Stack

- React 18
- TypeScript
- Vite
- TailwindCSS v4
- Framer Motion
- React Hook Form
- Lucide React
